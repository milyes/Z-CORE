NetSecurePro IA - Master Plan

Projet Cybersécurité Intelligent • Z-CORE
Date : Avril 2026 | Auteur : Mohammed Ilyes Zoubirou

🎯 Objectif Général

Développer un système de détection de menaces en temps réel basé sur l’analyse avancée de logs réseau, utilisant Snowflake comme plateforme principale et Snowpark pour le traitement scalable et sécurisé des données.

🔑 Fonctionnalités Clés

  • Enrichissement Géolocalisation : Conversion des IPs en données contextuelles (pays, ville, ASN, risque régional)
  • Détection Impossible Travel : Calcul de vitesse de déplacement entre deux connexions et détection d’anomalies (> 900 km/h)
  • Filtrage VPN vs Attaque : Distinction intelligente entre changement légitime de VPN et compromission de compte
  • Threat Intelligence Integration : Croisement avec des feeds d’IPs malveillantes
  • Feature Engineering Avancé : Entropie, fenêtres glissantes, séquences d’événements
  • Scoring de Risque Contextuel : Modèle IA qui combine toutes les features pour générer un score de menace

🛠️ Architecture Technique

Plateforme Principale

  • Snowflake Data Cloud (Snowpark Python + UDFs)
  • Tables : RAW_LOGS → ENRICHED_LOGS → FEATURE_STORE → INFERENCE_RESULTS

Composants Clés

  • Ingestion : Logs réseau (NetFlow, Firewall, SIEM) chargés via Snowpipe ou COPY
  • Enrichissement : UDFs pour Géolocalisation (MaxMind GeoIP2)
  • Feature Engineering : Window Functions + UDFs pour Impossible Travel et entropie
  • Modèle IA : Modèle d’anomalie (Isolation Forest / Autoencoder) ou classification supervisée
  • Output : Alertes en temps réel vers SIEM ou système de blocage
