# Z-CORE

**Stack privée auto-hébergée Zero Trust avec IA souveraine et module médical intégré**

> Souveraineté numérique maximale • Données critiques 100% locales • Never Trust, Always Verify

---

## Philosophie

Z-CORE est une infrastructure **hybride et consciente** construite par un développeur IA solo.

- **Cœur souverain** : Toutes les données critiques (mails, fichiers, identités, données médicales, modèles d’inférence) restent sur des serveurs contrôlés.
- **Assistants cloud** : Claude (API + Code) et Azure CLI sont utilisés uniquement comme outils d’aide au développement, jamais comme dépositaires de données.

**Objectif** : Maximiser la souveraineté tout en restant productif.
