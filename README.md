# Z-CORE

**Zero Trust Sovereign Stack**  
Stack privée auto-hébergée avec IA souveraine, SSO unifié et module médical intégré.

> Souveraineté numérique maximale • Never Trust, Always Verify • Données critiques 100% locales

---

## Philosophie

Z-CORE est une infrastructure **hybride et consciente** conçue par un développeur IA solo.

- **Cœur souverain** : Toutes les données sensibles (identités, mails, fichiers, données médicales, modèles d’inférence) restent sur des serveurs contrôlés.
- **Assistants cloud** : Claude API, Claude Code et Azure CLI sont utilisés uniquement comme outils d’aide au développement, jamais comme dépositaires de données.

**Objectif** : Maximiser la souveraineté tout en restant hautement productif.
